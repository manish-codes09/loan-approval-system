# 🏦 Loan Approval Prediction System

A Machine Learning web application that predicts whether a loan application will be approved based on applicant details. Built using **Python**, **Scikit-learn**, and **Streamlit**.

## 🚀 Features

* Predict Loan Approval Status
* Approval & Rejection Probability
* Interactive Streamlit Interface
* Fast Real-Time Predictions

 ## live demo
 https://loan-approval-system-acl45zybhluzfpu53wc9sl.streamlit.app/

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* NumPy
* Pickle



## 📊 Input Features

* Credit Score
* Applicant Income
* Loan Amount
* DTI Ratio
* Loan Term
